# AI能力一覧（2026-04-24 時点）

---

## 路選択（どの路から撒くか）

| 難易度                              | アルゴリズム                                                                                      |
| ----------------------------------- | ------------------------------------------------------------------------------------------------- |
| **easy（弱い）**                    | 完全ランダム                                                                                      |
| **normal（普通）**                  | ぐるぐる/ざくざく技が出る路を優先（ランダム選択）→ なければ賽壇（pit11）に近い順（pit10→9→8→7→6） |
| **hard（強い）**                    | ちらちら3回消費まではpit5着地を最優先 → 完了後はぐるぐる連鎖を3手先読みして最大化                 |
| **oni（鬼）/ oni-sente / oni-gote** | OniV2（DEFAULT_PARAMSで以下の要素を統合評価）                                                     |
| **robo（ロボ）**                    | OniV2（roboパラメータで以下の要素を統合評価）                                                     |

## OniV2 統合評価（鬼・ロボ共通の路選択スコアリング）

全路に対してスコア計算し最高得点の路を選ぶ。

| 評価要素                         | 内容                                                                                                               |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| **2手先読み**                    | 自分が撒いた後プレイヤーが最善応手した盤面を予測し、その後のAI側有利度ボーナス・プレイヤー脅威ペナルティを加算     |
| **脅威成長ペナルティ**           | 撒き後にプレイヤーの攻撃機会（ぐるぐる・ざくざく可能手）が増える場合に減点                                         |
| **ぐるぐる**                     | pit11着地でぐるぐる連鎖を評価。連鎖数²×乗数でボーナス。自路色整理（2色以下）でさらに加点                           |
| **ちらちら/ぽいぽい準備**        | pit5着地の価値（1〜3回目・序盤/中盤で閾値変化）                                                                    |
| **ざくざく**                     | 自路空き穴着地 + 対面に石がある場合に石数比例ボーナス。知識色（推測/自占い/確認プラス）が奪える場合に追加加点      |
| **ざくざく防御**                 | 撒き元の自路と対面のプレイヤー路に石があると非線形ペナルティ（被ざくざくリスク）                                   |
| **被ざくざく露出防止**           | 撒き後に自路の石が多くプレイヤー対面が空だと大ペナルティ                                                           |
| **プレイヤーぐるぐる破壊**       | 撒くことで既存のプレイヤーぐるぐる準備路を消せたらボーナス                                                         |
| **ちらちら被弾防止**             | 撒き後に相手路からpit11への着地可能穴が増えたらペナルティ                                                          |
| **くたくた妨害**                 | プレイヤーがくたくた発動可能な状況では相手路への石送り込みにペナルティ                                             |
| **石の色評価（pit11着地）**      | 推測プレイヤー占い色 > 自分の占い色 > 確認済みプラス中央石の優先度でボーナス。確定マイナス石・回避色に大ペナルティ |
| **路の色品質事前評価**           | 路全体に自占い/推測/確認プラス色が多いと選択前加点。確定マイナス色に減点                                           |
| **自路着地石の将来価値**         | 自路に落ちる石の色が知識ベース高価値なら加点。マイナス確定/回避色は減点                                            |
| **相手賽壇への高価値石流入防止** | 推測プレイヤー占い色・確認プラス色が相手賽壇（pit5）に流れ込む手を大減点                                           |
| **気分システム**                 | 毎ターンランダムで占い重視度（×0.3〜1.7）をゆらがせ、行動パターンを読まれにくくする                                |
| **防御タイブレーカー**           | 攻撃スコア差が閾値以内の手に限り防御ペナルティを加味して最終選択                                                   |

## 撒き順最適化（石を撒く順番）

| 難易度                | 内容                                                                  |
| --------------------- | --------------------------------------------------------------------- |
| **easy / normal**     | なし（元の順番のまま）                                                |
| **hard / oni / robo** | pit11（AI賽壇）に高価値石を、pit5（相手賽壇）に低価値石を優先送り込む |

**石の価値優先度：** 推測プレイヤー占い色 > 自占い色 > 確認プラス色 > 確定マイナス（pit11に絶対入れない）

## ざくざく後の石配置

| 難易度         | 内容                                                                                                        |
| -------------- | ----------------------------------------------------------------------------------------------------------- |
| **easy**       | ランダム                                                                                                    |
| **normal**     | 自占い色ならぐるぐるセットアップ路へ優先 → それ以外は非空路ランダム                                         |
| **hard**       | ぐるぐるセットアップ路を優先（被ざくざくリスクの低い路をさらに優先）                                        |
| **oni / robo** | スコア評価：ぐるぐるセットアップ路を基本優先 + 知識色（推測占い・確認プラス・cancel枚数）を加味して最適配置 |

## ちらちら/ぽいぽい判断

| 難易度                   | 内容                                                                               |
| ------------------------ | ---------------------------------------------------------------------------------- |
| **easy / normal / hard** | ちらちら残回数があれば常にちらちら。使い切ったらぽいぽい                           |
| **oni / robo**           | 自陣3石以上かつpeeksDone<2なら強制ちらちら。それ以外はぽいぽいの価値と比較して選択 |

**ぽいぽい判断の閾値（鬼）：**

- ちらちら残≥2 のとき閾値 = 25
- ちらちら残1 のとき閾値 = 6
- 推測色が相手賽壇にある = ぽいぽい価値22、石2個以上 = 4

## ぽいぽい石選択（全難易度共通）

| 優先度              | 対象                                                                       |
| ------------------- | -------------------------------------------------------------------------- |
| 1位（除去価値最大） | AI自身の占い色（相手にとって+5点の石）                                     |
| 2位                 | 推測プレイヤー占い色（+3点を奪う）                                         |
| 3位                 | AIが確認済みの中央プラス石（+1を奪う）                                     |
| 4位                 | プレイヤーが確認済みの中央プラス石                                         |
| 絶対除去しない      | 確定マイナス石（取ると相手のマイナスが消える）                             |
| 特別判断            | 自賽壇に確定マイナス石があれば相手賽壇より自賽壇を捨てる方が得な場合がある |

## memoシステム（プレイヤー占い色の推測）

**全難易度共通**でターン開始時に更新（ぽいぽい石選択・予測フェーズに使用）。

- プレイヤー路（pit0〜4）の石 ×1 + プレイヤー賽壇（pit5）の石 ×3 で色頻度を累積
- AIが確認済みの中央石の色を除外（それは個人占い色ではないため）
- 累積頻度最多 → `inferredPlayerColor`（推測プレイヤー占い色）
- 累積頻度最少（3色以上データあり） → `playerAvoidedColor`（推定マイナス色）
